Rouse Sample Animation

Choose an anti-aliasing level and press "Play!" to view the animation.

The code for this is in this GitHub repository. The engine is called "rouse" because that's what happens if you look for synonyms of "animate".

There's some keys to control this (sorry mobile users, I'll add touchy buttons some time):

  • R to reload the scene
  • F to toggle the fps counter
  • W for fullscreen mode
  • Q to leave fullscreen
  • D for the pivot point debug view
